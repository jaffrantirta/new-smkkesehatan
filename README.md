# Badewa Jaffran Theme

**Version:** 1.0
**Author:** Jaffran Tirta Artika
**Author Website:** https://github.com/jaffran

## Description

A modern, professional WordPress theme designed for educational institutions and schools. Developed by Jaffran Tirta Artika, a software developer from ITM Bali. Perfect for vocational schools, colleges, universities, training centers, and educational organizations.

## Features

- ✅ Fully Responsive Design
- ✅ Customizable Color Scheme (6 color options)
- ✅ Dynamic Kompetensi Section (1-10 program carousel)
- ✅ Dynamic Keunggulan Section (1-8 items)
- ✅ Floating WhatsApp Button (customizable position & message)
- ✅ Smooth Scroll Animations
- ✅ Custom Page Templates (About, Contact, FAQ, Facilities)
- ✅ WordPress Customizer Integration
- ✅ Bootstrap 5.3.3 Framework
- ✅ Modern Typography (Playfair Display + Source Sans 3)
- ✅ SEO Friendly
- ✅ Translation Ready
- ✅ WhatsApp Integration
- ✅ Social Media Links
- ✅ Google Maps Integration

## Installation

1. Download the theme folder
2. Upload to `/wp-content/themes/` directory
3. Activate the theme in WordPress Admin → Appearance → Themes
4. Go to Appearance → Customize to configure theme settings

## Customization

### Theme Colors
Navigate to **Appearance → Customize → Theme Colors** to customize:
- Primary Color
- Primary Dark Color
- Accent Color
- Cream/Background Color
- Ink/Text Color
- Muted Text Color

### Other Customization Options
- Blog Settings (Default Blog Image)
- Floating WhatsApp Button (Enable/Disable, Number, Message, Position)
- Header Settings (Phone, Email, Social Media)
- Hero Section
- Sambutan (Welcome Message)
- Kompetensi Keahlian/Program (Dynamic count 1-10 carousel items, customizable content)
- Keunggulan/Advantages (Dynamic count 1-8 items, customizable content)
- About Page Content
- FAQ Items
- Facilities
- Contact Information
- Sidebar Settings
- Footer Settings

## Page Templates

- **Front Page** - Homepage with hero, sambutan, programs, advantages, and blog
- **About Us** - Company profile with vision & mission
- **Contact Us** - Contact information with map and WhatsApp CTA
- **FAQ** - Frequently asked questions with accordion
- **Fasilitas** - Facilities showcase with alternating layout

## Theme Screenshot

To add a theme screenshot:
1. Create or capture a screenshot of your website homepage (1200x900px recommended)
2. Save it as `screenshot.png` in the theme root directory
3. The screenshot will appear in Appearance → Themes

You can use tools like:
- Browser screenshot tools (Full page)
- Photoshop/GIMP
- Online screenshot generators
